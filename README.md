Wealth IQ Calculator

An interactive financial health dashboard that converts monthly income
and recurring expenses into a visual financial score.

Features:
• Expense ratio and savings rate calculation
• Housing and transportation burden analysis
• Interactive charts for income vs expenses
• Real-time financial scoring system
• Calculation history tracking

Tech stack:
React • TypeScript • Recharts • Tailwind • Render deployment

Built as an independent portfolio project demonstrating financial analytics and interactive data visualization.
